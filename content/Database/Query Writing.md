## Getting Started

We need some tables,

```markdown

**Project**

|-------------------------------|
|   Project     |       Title   |
|-------------------------------|
|       P1      |   Project 1   |
|       P2      |   Project 2   |
|       P3      |   Project 3   |
|-------------------------------|

**Person**

|-------------------------------|
|   Person      |       Name    |
|-------------------------------|
|       P1      |   Project 1   |
|       P2      |   Project 2   |
|       P3      |   Project 3   |
|       P4      |   Project 4   |
|-------------------------------|


**Material**

|-------------------------------|
|   Material    |       Name    |
|-------------------------------|
|       M1      |   Material 1  |
|       M2      |   Material 2  |
|       M3      |   Material 3  |
|       M4      |   Material 4  |
|       M5      |   Material 5  |
|       M6      |   Material 6  |
|-------------------------------|


```

And you can generate the following table,

```markdown
|---------------------------------------------------------------------------|
|   Project     |   Person      |   Hours   |   Material    |   Quantity    |
|---------------------------------------------------------------------------|
|
|---------------------------------------------------------------------------|
```
